# 🎮 Mini Games Hub

Este projeto é uma página web simples criada para reunir e disponibilizar alguns dos meus jogos feitos com a Godot.

Os jogos foram exportados para HTML e podem ser jogados diretamente pelo navegador.

## 🌐 Sobre o site

O site consiste em uma única página com:

- Título e breve descrição
- Cards para cada jogo
- Botões que levam diretamente para os jogos

A proposta foi criar algo simples, funcional e visualmente agradável para compartilhar os projetos.

## 🎨 Estilo

O visual do site segue uma estética mais chamativa, com:

- Tema neon
- Animações simples em CSS
- Interações básicas (hover em cards e botões)

## 🎮 Jogos disponíveis

- Mini-Voley  
- pew-pew.io  
- Watch Your Head  
- Race ES  

## 📚 Observação

Este projeto foi desenvolvido como um site simples para organizar e compartilhar meus jogos.

O foco foi mais na apresentação e no visual do que em complexidade técnica.
